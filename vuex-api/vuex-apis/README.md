# vuex-apis

## vuex 执行过程
(组件发起（dispatch）action，action执行异步操作，异步操作执行成功提交到mutation，mutation当中来修改state，state发生改变组件随之更新)

vuex-persistedstate  初始化state中的数据到缓存中，可以让刷新数据不丢失
